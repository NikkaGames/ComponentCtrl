# ComponentCtrl

Minimal AW22XXX control client for Windows.

## Commands
- `ComponentCtrl info`
- `ComponentCtrl list-configs`
- `ComponentCtrl apply-config <configId> [rgbOverride]`
- `ComponentCtrl apply-effect <effectId> [rgbOverride]`
- `ComponentCtrl set-imax <imaxCode>`
- `ComponentCtrl init`
- `ComponentCtrl off`
- `ComponentCtrl read <page> <register>`
- `ComponentCtrl write <page> <register> <value>`

## Build
- Run `build_x64.cmd` in this directory.
